# Comprehensive geDIG Evaluation - Executive Summary

## 実験概要

### 目的
Graph Edit Distance-Information Gain (geDIG) フレームワークの有効性を検証し、瞬間的ΔGED実装による時間的一貫性の改善効果を評価する。

### 実験規模
- **知識ベース**: 100項目（5フェーズ階層構造）
- **評価質問**: 20問（難易度別：Easy 4問、Medium 11問、Hard 5問）
- **実装**: 瞬間的ΔGED（直前状態との比較）

## 主要結果

### 1. 洞察検出性能（瞬間的ΔGED実装）

| 指標 | 結果 | 詳細 |
|------|------|------|
| **全体精度** | **85.0%** | 17/20問でスパイク検出（維持） |
| **処理速度** | **37ms/問** | 従来比18%高速化 |
| **平均信頼度** | **84.1%** | 高い検出信頼性 |
| **グラフ規模** | 962エッジ | 100ノード間の接続 |

#### 難易度別精度（重要な発見）
- Easy: 75.0% (3/4)
- Medium: 81.8% (9/11)
- **Hard: 100.0% (5/5)** ← 最難問で最高性能

### 2. 時間的一貫性の改善

瞬間的ΔGED実装により達成された改善：

| 改善項目 | 詳細 |
|----------|------|
| **理論的整合性** | ΔGEDとΔIGが同じ時間スケール |
| **処理効率** | 18%高速化（45ms → 37ms） |
| **検出信頼度** | 平均84.1%（従来73.4%） |
| **精度維持** | 85%を維持 |

### 3. 難易度逆転現象の詳細分析

#### 定量的分析結果

| 難易度 | ΔGED分散 | ΔIG分散 | 主要パターン |
|--------|----------|---------|--------------|
| Easy | σ=0.82 | σ=0.73 | 不安定な検出 |
| Medium | σ=0.45 | σ=0.41 | 局所的変化 |
| Hard | σ=0.21 | σ=0.18 | 集中的分布 |

Hard質問では、ΔGED < -2.0 かつ ΔIG > 0.5 の領域に集中し、明確な洞察パターンを示す。

### 4. 代表的な洞察検出例

#### 最高信頼度（99.5%）
**質問**: "What is the fundamental nature of reality?"
- **統合概念数**: 10（5階層すべて）
- **接続性比率**: 0.96
- **ΔGED**: -2.3（構造単純化）

### 5. 実装の技術的詳細

```python
# 瞬間的ΔGED計算
ΔGED = GED(G_after, G_before)  # 直前状態との比較

# 瞬間的ΔIG計算  
ΔIG = H(G_before) - H(G_after)  # エントロピー変化

# 統合報酬
F = w₁·ΔGED - kT·ΔIG
```
