# 強化学習・迷路実験

InsightSpike-AIの強化学習機能と迷路探索能力をテストする実験群。

## 含まれる実験

### 強化学習比較実験
- `comprehensive_rl_showcase.py` - 包括的な強化学習デモンストレーション
- `enhanced_rl_comparison.py` - 拡張された強化学習比較実験
- `rl_comparison_experiment.py` - 基本的な強化学習比較
- `rl_maze_comparison.py` - 迷路環境での強化学習比較

### 迷路探索・可視化
- `maze_exploration_visualizer.py` - 迷路探索過程の可視化
- `maze_visualization_interactive.py` - インタラクティブな迷路可視化
- `realtime_maze_animation.py` - リアルタイム迷路アニメーション

## 実行方法

```bash
cd rl_experiments
python comprehensive_rl_showcase.py
```

## 注意事項

これらの実験は「迷路探索」と呼んでいますが、実際には：
- 複雑な推論プロセスの比喩的表現
- 認知科学実験（パラドックス解決等）
- 強化学習アルゴリズムの性能比較

が主な内容です。
