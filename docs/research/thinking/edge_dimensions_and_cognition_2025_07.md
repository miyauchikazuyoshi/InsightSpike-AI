# エッジ次元と認知メカニズムに関する思索（2025年7月26日）

## 1. 概念の自然な分離現象

「りんご美味しい」「りんご赤い」など、りんごに関する様々なセンテンスを多次元の類似度で評価していくと、「りんご」のみが自然に分離されていく現象。

```
多次元評価により：
- 意味的類似度: りんご成分が共通
- 統語的パターン: 異なる（形容詞、動詞、修飾）
- 時間的共起: 文脈依存
- 感情極性: 異なる

→ 「りんご」という概念核が浮かび上がる
```

これは人間の概念形成プロセスに非常に近い。

## 2. 生命の認知発達とエッジ次元

### 基本仮説
生命の埋め込みエッジは基本的に5感で捉えられる感覚器がデフォルト。あるタイミングからエッジ埋め込み次元が増えていく。

### 発達段階
1. **初期**: 5感ベースの直接的な関係性
   - 視覚的類似（形、色）
   - 聴覚的パターン（音の高低）
   - 触覚的性質（硬い、柔らかい）

2. **発達後**: 抽象的な関係性を獲得
   - 因果関係
   - 時間的順序
   - 社会的意味
   - 感情的価値

### ウイスキーの味覚マップの例
バーテンダーの「味蕾が育ってない」という指摘。

初心者: `[アルコール感, 甘い/辛い]`（2次元）
経験者: `[アルコール感, 甘い/辛い, ピート香, 樽の香り, 余韻の長さ, ...]`（多次元）

## 3. geDIGの極小主義

### GED評価もメッセージパッシングも極小でしか使わない

従来のGNN: 「グラフ全体を活用」→ 計算爆発
geDIG: 「瞬間的な洞察に必要な最小限」→ 人間の直感に近い

### 1-2hop限定の恩恵
```
実際にアクセスされるエッジ数:
1-hop: 候補5個 × 隣接20 = 100エッジ
2-hop: 100 × 20 = 2,000エッジ
合計: 2,100エッジのみ！

→ 50次元でも余裕（420KB程度）
```

## 4. 共感覚とマジックナンバー7

### 共感覚 = 多次元エッジの「誤配線」？
```
通常: 視覚 → 視覚野のみ
共感覚: 視覚 → 視覚野 + 味覚野（クロストーク）
```

「黄色いものを見たら、しょっぱい味がする」

### マジックナンバー7 = 並列エッジ評価の限界

同時に評価できるエッジ本数の限界が7±2。

```
「りんご」から同時に連想：
→ 赤い、果物、甘い、丸い、木になる、ニュートン、青森
（7つが限界、8つ目は順次処理）
```
