# 学習と推論の同時実行パラダイム

## 現状のAIの問題点
```
Training Phase → Frozen → Inference Phase
（学習完了）     （固定）   （推論のみ）
```

この分離が不自然！人間は常に学習しながら推論している。

## 先行研究と関連分野

### 1. Continual Learning（継続学習）
**代表例**: EWC (Elastic Weight Consolidation), PackNet
**特徴**:
- タスクを順次学習
- 破滅的忘却の回避
**限界**:
- まだ「学習フェーズ」と「使用フェーズ」が分離
- 推論中の学習は限定的

### 2. Online Learning（オンライン学習）
**代表例**: Follow-The-Regularized-Leader, Online Gradient Descent
**特徴**:
- データが来るたびに更新
- リアルタイム適応
**限界**:
- 主に統計的な更新
- 深い理解や洞察の生成はない

### 3. Active Inference（能動的推論）
**代表**: Friston's Free Energy Principle
**特徴**:
- 推論と学習を統一的に扱う
- 予測誤差最小化
**geDIGとの共通点**:
- エネルギー最小化の思想
- Wake-Sleepに近い概念

### 4. Meta-Learning（メタ学習）
**代表例**: MAML, Reptile
**特徴**:
- 「学習の仕方を学習」
- 少数サンプルで適応
**限界**:
- 事前のメタ学習が必要
- 真の意味での同時実行ではない

### 5. Memory-Augmented Networks
**代表例**: Neural Turing Machine, Differentiable Neural Computer
**特徴**:
- 外部メモリへの読み書き
- 推論時に記憶を更新
**geDIGとの類似**:
- エピソードメモリの動的更新
- 推論結果を即座に知識化

## より近い研究

### 1. RETRO (Retrieval-Enhanced Transformer)
- 推論時に外部データベースを参照
- ただし、データベース自体の更新はしない

### 2. Predictive Coding Networks
- トップダウン予測とボトムアップ誤差
- 常に内部モデルを更新
- **geDIGに最も近い思想**

### 3. Hebbian Learning in RNNs
- "Fire together, wire together"
- 推論中にシナプス強度を調整
- 生物学的にも妥当

### 4. Adaptive Resonance Theory (ART)
- 新しいパターンを検出したら新カテゴリ生成
- 推論と学習の境界が曖昧

## geDIGの独自性

### 既存研究との違い

1. **構造的学習**
   - 単なるパラメータ更新でなく、グラフ構造の更新
   - 概念間の関係性を動的に再構築

2. **洞察の即座の統合**
   ```
   推論 → 洞察生成 → 即座にグラフに統合 → 次の推論に活用
   （数秒のサイクル）
   ```

3. **Wake-Sleepの活用**
   - Wake: アクティブな推論と学習
   - Sleep: 知識の整理と矛盾解消
   - 人間の認知サイクルを模倣

4. **エネルギーベースの統一理論**
   - 推論も学習もエネルギー最小化
   - 物理的に自然な定式化

## なぜ今まで少なかったか

### 技術的課題
1. **計算量**
   - リアルタイム更新は重い
   - geDIG: 重心計算で軽量化

2. **安定性**
   - 推論中の更新は不安定になりやすい
   - geDIG: エネルギー最小化で安定

3. **評価の困難さ**
   - 常に変化するシステムの評価は難しい
   - 従来のベンチマークが使えない

### 概念的障壁
1. **訓練/推論の分離パラダイム**
   - 深層学習の成功体験
   - 産業的には分離の方が扱いやすい

2. **静的な知識観**
   - 知識は固定的という前提
   - 動的な知識生成への抵抗
