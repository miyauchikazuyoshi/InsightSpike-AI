- **poetry.lock 再生成**: 新しい依存関係構成で互換性確保
- **CI設定簡素化**: 複雑な手動 pip インストールを削除

### 📊 修正内容詳細
```toml
# 修正前（問題あり）
faiss-gpu = [
  { version = ">=1.7.0", markers = "sys_platform != 'darwin'" }
]

# 修正後（CI対応）
# Core ML dependencies - CPU-only for compatibility
faiss-cpu = ">=1.11.0"

# Optional dependencies for enhanced functionality
torch-geometric = { version = "==2.4.0", optional = true }
```
