# 分析ツール

InsightSpike-AIの性能分析とイノベーション評価ツール。

## 含まれるツール

### イノベーション分析
- `innovation_analysis.py` - 技術革新度の定量的分析
- `robotics_spatial_experiment.py` - ロボティクス・空間認知実験

## 用途

- 技術的性能の客観的評価
- 学術論文用データ生成
- 特許技術の定量的証明

## 分析メトリクス

- ΔGED (Generalized Edit Distance) - 認知的変化量
- ΔIG (Information Gain) - 情報獲得量
- 処理レイテンシ・スループット分析
