# InsightSpike-AI: 創発的知識発見システム

## 🎯 コアバリュー：RAG性能ではなく「創発性」

### 従来のRAGとの根本的な違い

#### 一般的なRAG
- 検索精度の向上が目標
- 「聞かれたことに正確に答える」
- 既存知識の効率的な取り出し
- メトリクス：Recall@K、MRR、F1スコア

#### InsightSpike-AI
- **知識の創発的発見が目標**
- 「教えられていないことを発見する」
- 新しい知識関係の自律的創造
- メトリクス：クロスジャンル洞察率（78.4%達成！）

